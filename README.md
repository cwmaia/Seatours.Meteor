Seatours.Meteor
===============

How Deploy
----------

On Seatours.Meteor directory:

`meteor bundle ../bundle.tgz`

> If Seatours.Meteor is a recent clone do a `mrt` before bundle it.

Go to bundle.tgz directory and:

`tar -zxvf bundle.tgz`

Restart the machine or:

`service meteor restart`

sudo may be needed.
